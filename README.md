# Azure Project Submission

### 1. Selected Region
I chose the South Africa region because it provides the lowest geographic network latency for users like me on the African continent (Nigeria), ensures local data residency compliance, and optimizes performance for resources deployed within this zone.

### 2. Shared Responsibility Model
Under the cloud Shared Responsibility Model, Microsoft manages the physical data center infrastructure, physical security, and core hardware virtualization.

I am responsible for managing:
* **Data & Information:** Ensuring proper data classification, retention policies, and encryption settings for any data stored within my resources.
* **Identity & Access Management:** Assigning user permissions using Role-Based Access Control (RBAC), managing authentication credentials, and enforcing the principle of least privilege.
* **Configuration Governance:** Setting up and monitoring active cloud resource configurations, establishing budgets, and configuring tracking alerts to monitor cost and usage efficiency.
